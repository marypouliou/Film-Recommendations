Title: Film Recommendations Based on your Mood

Description: 

A web application that displays content based on what mood users state they are in for. Content is retrieved based on moving multiple sliders and, therefore, a profile based on multiple moods is created.

Content is uploaded in xml format

The algorhithm has been annotated with comments throughout.

Author: Maria Pouliou
Date created: 18th November 2017 

Browser used: Google Chrome for Mac, Version 62.0.3202.94 (Official Build) (64-bit)
